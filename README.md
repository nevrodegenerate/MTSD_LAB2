# Опис застосунка
Консольний застосунок, що приймає шлях до текстового файлу та перетворює його Markdown розмітку в html або ANSI. 

# Як запустити
На ОС Windows з підтримкою C++, відкрити MTSD_LAB1\x64\Debug\MTSD_LAB2.exe в bash або командній строці Windows, вказати шлях до вхідного файлу та опціональні аргументи.

# Інструкція використання
При запуску необхідно вказати принаймні один аргумент (шлях до вхідного файлу), після чого програма виведе вихідний текст в консолі. 
Якщо після першого аргумента, вказано аргумент --out та шлях та назву вихідного файлу, натомість текст не виведеться та буде створено новий файл.
Якщо після першого аргумента або шляху та назви вихідного файлу, вказано аргумент "--format=ansi", то розмітку буде переведено в ANSI,
а якщо аргументом буде "--format=html", то розмітка буде переведена в html.
Якщо аргумент "--format=" буде відсутнім, то при виводі в консоль, розмітка буде переведена в ANSI, а при виводі у файл - в html.
